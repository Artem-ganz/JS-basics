Використання JavaScript на веб-сторінках

Клієнтський JavaScript-код може вбудовуватися в HTML-документи чотирма способами:

  • вбудовані сценарії між парою тегів script та /script;
  • із зовнішнього файлу, заданого атрибутом src тега script;
  • в обробник події, заданий як значення HTML-атрибута, такого як onclick або onmouseover;
  • як тіло URL-адреси, що використовує спеціальний специфікатор псевдопротоколу JavaScript:.
JavaScript - Мультипарадигмова мова програмування. Підтримує об'єктно-орієнтований, імперативний та функціональний стилі. Є реалізацією специфікації ECMAScript (стандарт ECMA-262 [9]).

JavaScript зазвичай використовується як вбудована мова для програмного доступу до об'єктів додатків. Найширше застосування знаходить у браузерах як мову сценаріїв для надання інтерактивності веб-сторінкам

Основні архітектурні риси: динамічна типізація, слабка типізація, автоматичне керування пам'яттю, прототипне програмування, функції як об'єкти першого класу.
